AROS currently has 3 prefs configuration sets. Please try to
keep changes relevant to the respective set.

classic
====

The classic set is configured to run on more modest/classic hardware.
Settings are mostly Amiga like, and configured for low resolution
display, and low power hardware.

default
=====

The default set is configured for more modern hardware, with settings
more suitable for display on higher resolution displays and with more
demanding hardware requirements.

showcase
======

The showcase set is configured to enable most of the features and options
exposed by the AROS subsystems, with prefs enabling options not normally
enabled by default. This set is also configured for more powerfull hardware
configurations.
